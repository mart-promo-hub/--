# 🚀 منصة الترويج الذكي - Smart Promo Hub

منصة ترويج ذكية تعمل بالذكاء الاصطناعي لشبكة Pi Network Mainnet.

---

## ✨ الميزات

- **🤖 توليد محتوى بالذكاء الاصطناعي** – نصوص إعلانية احترافية فورية.
- **🌐 دعم 5 لغات** – واجهة متعددة اللغات.
- **💰 مدفوعات Pi Network** – تكامل مع عملة Pi الرقمية.
- **📱 تكامل مع منصات التواصل الاجتماعي** – فيسبوك، إنستغرام، تويتر، يوتيوب، تيك توك، لينكدإن، سناب شات، واتساب، تيليجرام، بينترست.
- **📊 لوحة تحكم تفاعلية** – إحصائيات فورية عن الحملات.
- **📝 إدارة الحملات الإعلانية** – إنشاء، تعديل، حذف، وعرض النتائج.
- **📈 تقارير الأداء** – مشاهدات، نقرات، تفاعل، تحويلات، وعائد الاستثمار (ROI).
